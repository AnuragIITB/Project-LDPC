\section{ Notations }
\begin{frame}[t]
\frametitle{ Notations }
\begin{itemize}
\item n - Block size.
\item Eb/No - Input SNR in db to maintain input BER between $10^{-2}$ to $10^{-3}$.
\item BER - output bit error rate.
\item CDB - Number of correctly decoded blocks.  
\item T - Average time to decode a code block in seconds.
\item Itr - Average number of iterations per block.
\end{itemize}
\end{frame}

\begin{frame}[t] 
\frametitle{ Conclusion }
\begin{itemize}
\item All the specification are matched for the rates upto 0.9.
\item The specs are not met for rate = 0.95. The reason is use of random matrix. As number of rows in matrix
starts decreasing the cycles starts building thus decreasing girth of the the graph. 
\end{itemize}


\end{frame}
